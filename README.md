# 🐱 Social Media Cats

[![GitHub Pages](https://img.shields.io/badge/-DEMO%20SITE-222?style=for-the-badge&logo=github)](https://alexh73.github.io/social_media_cats/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Социальная сеть для владельцев кошек с возможностью публикации фото, лайков и комментариев.

![Главный экран приложения](./images/image.png "Пример интерфейса")

## ✨ Особенности
- Просмотр ленты с котиками
- Загрузка новых фото питомцев
- Система лайков и комментариев
- Адаптивный интерфейс
- Анимации взаимодействия

## 🛠 Технологии
- HTML5
- CSS3
- CSS Modules
- GitHub Pages (деплой)

## 🚀 Запуск проекта

### Установка
```bash
git clone https://github.com/AlexH73/social_media_cats.git
cd social_media_cats
npm install
```
